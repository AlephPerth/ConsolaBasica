Comandos basicos:

Comando cd

Sirve para navegar por los archivos y directorios. La sintaxis de 'cd /directorio/subdirectorio' o estando dentro del directorio raiz cd 'subdirectorio/subdirectorio'.

Por ejemplo: supongamos que estamos en /home/Hernan/Documentos y quiero ir a Fotos que es un subdirectorio de Documentos. Para hacerlo, simplemente escribo cd Fotos.

Si se deseo ir a un directorio completamente nuevo debo escribir cd seguido de la ruta completa del directorio: cd /home/ Hernan/Peliculas.

Algunos atajos:

  • cd .. (con dos puntos) para ir un directorio hacia arriba, de cd /home/ Hernan/Peliculas. a cd /home/ Hernan.
  • cd / para ir directamente a la carpeta de inicio
  • cd- (con un guión) para ir al directorio anterior

    • Comando ls

    El comando ls se usa para ver el contenido de un directorio. Por defecto, este comando muestra el contenido del directorio donde se esta actualmente.

    Si se desea ver el contenido de otro directorio, se debe escrir ls y luego la ruta del directorio. Por ejemplo: 'ls/home/Hernan/Documentos'.

    Algunos atajos:

  • ls -R también listará todos los archivos en los subdirectorios
  • ls -a mostrará los archivos ocultos
  • ls -l listará los archivos y directorios con información detallada como los permisos, el tamaño, el propietario, etc.
  • ls -h lista en "modo humano" es decir muestra el peso de los archivos y directorios en MB en vez de bits

    • Comando cat

    cat se utiliza para mostrar el contenido de un archivo en pantalla. Para ejecutar este comando, escribe cat seguido del nombre del archivo y su extensión. Por ejemplo: cat archivo.txt.

    Comando pwd

    El comando devuelve la ruta completa de donde nos encotramos en ese momento

    Comando grep

    Permite buscar dentro del texto en un archivo caracteres, palabras (y numeros) u oraciones. La sintaxis es grep 'palabra buscada' NombreDelArchivoDondeLoBusco.SuExtencion. Por ejemplo: grep 'azul' notepad.txt busca la palabra azul en el archivo de nombre notepad de extencion .txt. Las líneas que contienen la palabra buscada se mostrarán en pantalla.

    Un tip es usar el * delante de la extencion del archivo (txt, jpg, png, etc.) el * funciona como comodin, es decir va a buscar en todos los archivos que tengan esa extencion dentro del directorio actual

    Comando find

    Permite buscar archivos por su nombre dentro del directorio actual. La sintaxis es find NombreDelArchivoQueBusco.SuExtencion. Por ejemplo: find notepad.txt buscara el archivo de nombre notepad de extencion .txt dentro del directorio y se mostrarán en pantalla en caso de encontrar algo.

    Comando df

    Nos devuelve un informe del uso del espacio en disco del sistema que se muestra en porcentaje y KB. Si se desea ver el informe en megabytes, se usa df -m.

    Comando du

    Nos devuelve el peso de archivo o un directorio, si se desea ver el informe en bytes, kilobytes y megabytes, se usa du -h

    Comando top

    Es equivalente al Administrador de tareas en Windows, el comando top mostrará una lista de los procesos en ejecución y la cantidad de CPU que utiliza cada proceso.

    Comando head

    Se usa para ver las primeras líneas de cualquier archivo. De manera predeterminada, mostrará las primeras diez líneas, pero se puede cambiar este número utilizando -n, por ejemplo: head -n 5 NombreDeArchivo.extencion nos devolvera las primeras 5 lineas del archivo

    Comando tail

    Se usa para ver las ultimas líneas de cualquier archivo. De manera predeterminada, mostrará las ultimas diez líneas, pero se puede cambiar este número utilizando -n, por ejemplo: tail -n 5 NombreDeArchivo.extencion nos devolvera las ultimas 5 lineas del archivo

    Comando rm -rf

    Nunca de los jamases usen este comando, es muerte y destruccion

    Como nota al margen, el shell de Linux distingue entre mayúsculas y minúsculas. Por lo tanto, debes escribir el nombre del directorio de forma exacta.

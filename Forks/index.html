Comandos basicos

COMANDO ¿Que hace?
cd Permite moverse entre directorios
ls Lista el contenido de un directorio
cat Nos permite ver y editar el contenido de un archivo
pwd Nos dice la ruta de donde nos encontramos en este momento
clear Limpia la pantalla, funciona de igual manera con ctrl+L
grep Busca archivos y directorios que contengan la palabra/s buscadas
grep Busca archivos y directorios
df Muestra espacio en disco
du Muestra el peso de un archivo o directorio
top Similar al administrador de tareas de Windows
head Nos permite ver las primeras líneas de un archivo de texto
tail Nos permite ver las ultimas líneas de un archivo de texto
rm -rf ***JAMAS USAR ESTE COMANDO***
